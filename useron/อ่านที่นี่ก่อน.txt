ทำ User Online สามารถเช็คได้ว่ามีคน Online อยู่ที่หน้าไหนบ้าง

*วิธีการติดตั้ง*

1. ใช้โปรแกรม Notepad (อยู่ที่ Start -> All Programs -> Accessories -> Notepad) เปิด File ชื่อ online.php ครับ
2. ให้ไปแก้ไข CODE ในไฟล์นี้ครับ (ที่สำคัญจะอธิบายเป็นภาษาไทย)
3. แก้เสร็จแล้วก็ Save ครับ (ไปที่ File --> Save นะ)
4. เสร็จแล้วให้ UPLOAD ไฟล์ทั้งหมดขึ้นไปใน SERVER ที่คุณใช้เลยครับ
5. ใช้โปรแกรม FTP ของคุณทำการ เปลี่ยนโหมดไฟล์ - Folder (CHMOD) ของไฟล์ดังต่อไปนี้ (สำหรับ Unix/Linux) ครับ สำคัญมากครับ 

      ไฟล์ - Folder ที่ต้องเปลี่ยน		CHMOD เป็น
               online.html			       666
               online.php			       755
                 online.txt			       777
                 record.txt			       777
               นอกเหนือจากนี้		               ปล่อยไว้ไม่ต้องเปลี่ยน

6. และหากคุณต้องการให้แสดง ข้อความแสดง User Online ที่ไหน ก็นำ Code ด้านล่างไปไว้ที่นั่นครับ <body> นะครับ (ดูตัวอย่างที่ไฟล์ test1.htm หรือ test2.htm)


<font face="MS Sans Serif" size="2" color="#008000">
<!-- ใส่  URL ของไฟล์ online.php?js=1 ครับ -->
<script src="http://www.thamwebsite.com/test/useron/online.php?js=1"></script>
</font>


7. และหากว่าคุณต้องให้มันเช็คว่าหน้าไหนมีคน Online เท่าไร ก็ให้คุณนำ Code ในข้อ 8 ไปไว้ในหน้านั้นนะครับ
8. ขออธิบายการทำงานหน่อยนะครับ เช่น คุณนำ Code ในข้อ 8 ไปไว้ใน 2 หน้า ซึ่งมีชื่อว่า test1.htm กับ test2.htm และ ในขณะนั้นหากมีคนเปิด

         หน้า test1.htm แล้ว ในหน้ารายงานผล จะแสดงว่า มีคน Online 1 คนในหน้า test1.htm
         หน้า test1.htm และ test2.htm พร้อมกัน ในหน้ารายงานผล จะแสดงว่า มีคน Online 1 คนในหน้า test1.htm และอีก 1 คนในหน้า test2.htm
         หน้า test1.htm พร้อมกัน 2 คน ในหน้ารายงานผล จะแสดงว่า มีคน Online 2 คนในหน้า test1.htm
         หากคนเดียวกันหละ แต่เปิดทั้ง 2 หน้าพร้อมกัน ในหน้ารายงานผล จะแสดงว่ามีคน Online 1 คนในหน้าที่คนนั้นเปิดล่าสุด

9. เข้าหน้า รายงานหน้าที่มีผู้เยี่ยมชม ได้โดยการเปิดหน้า online.php (หรือคลิ๊กที่ LINK ตรงข้อความแสดง User Online ก็ได้)
10. ใครมีข้อสงสัยอะไรก็อีเมล์มาถามกันได้นะครับที่ au_cty@hotmail.com ครับ
